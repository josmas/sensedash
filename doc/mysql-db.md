# MySQL setup

In MySQL you need to define database config in config/config.js

```console
"mysql_ip": "ipaddresshere",
"mysql_port": "portnumberhere",
"mysql_user": "username",
"mysql_pass": "passwordhere",
"mysql_database": "databasename",
```
